# ip adress config

## Configure ip adresses

```text
Configure terminal
interface <interface>
ip address <ip address> <subnet>
no shutdown
exit
```

> `Sutdown` or `no shutdown` disabeld or enabelde the port
>
> show ip interface brief

### interfaces

Serial connection 

```text
interface S0/0/0
```

Gigabit eternet connection

```text
interface G0/0
```

Loopback interface

```text
interface loopback 0
```
